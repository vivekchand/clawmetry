## Usage

1. **Pick a template** that matches your automation pattern
2. **Copy to your skills directory**: `cp skill-templates/simple-api-wrapper.md my-skills/my-service/SKILL.md`
3. **Follow the customization checklist** in the template
4. **Test locally** before deploying

## Template Structure

Each template includes:
- Complete SKILL.md example with metadata
- Working code examples (bash scripts, Python, etc.)
- Configuration templates (JSON, environment variables)
- Customization checklist
- Dependencies and installation guide
- Production deployment considerations
