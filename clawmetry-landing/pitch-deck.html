The Solution

ClawMetry: full visibility, anywhere

ClawMetry gives you a real-time dashboard of everything your AI agents are doing — token costs, live flow visualization, tool call history, memory browser, sub-agent trees. Open source locally, cloud-synced with E2E encryption for anywhere access.
⚡

Flow Tab — Live agent visualization

Watch every tool call, sub-agent spawn, and message in real time as your agent works

🧠

Brain Tab — Full thought and action transcript

Every tool call, decision, and output — searchable, filterable, timestamped

💰

Tokens Tab — Per-session cost breakdown

Know exactly how much each session costs before your bill arrives

🔐

E2E Encrypted Cloud Sync

Data encrypted on your machine before upload — only you can read it in the browser
