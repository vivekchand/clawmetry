How it works

Three steps to full observability inside your NemoClaw environment.

01

Run on the host machine

One command on the host running NemoClaw. ClawMetry installs across all your NemoClaw instances and applies the ClawMetry sync policy automatically.

curl -fsSL https://clawmetry.com/install.sh | bash
02

E2E encrypted cloud sync

An encryption key is generated on your host. Only the encrypted telemetry stream reaches ClawMetry Cloud — prompts, responses, and API keys never leave your machine.

clawmetry onboard
03

Watch from any browser

Open app.clawmetry.com. See live agent sessions, token spend per call, memory file diffs, and cron job status — all in real time from your phone or laptop.

app.clawmetry.com
