Common questions

Why is the self-hosted version fully free?
Because we believe observability tooling shouldn't be a moat. ClawMetry is MIT licensed — fork it, modify it, run it forever. We make money from the cloud relay, not by locking features. This also means the OSS community helps us build better features for everyone.
What exactly does Cloud Pro unlock?
Remote access. That's it. When you run ClawMetry locally, it's accessible only on your LAN. Cloud Pro adds a secure relay so you can open app.clawmetry.com from your phone, another laptop, or anywhere — and see your agents working in real time. All features are identical.
Is my data safe in the cloud?
Yes. Your data is encrypted on your machine before it ever leaves — we use AES-256-GCM client-side encryption. The cloud server stores only ciphertext. We can't read your agent data even if we wanted to.
What counts as a node?
One node = one machine running OpenClaw with the sync daemon. If you have a Mac, a home server, and a VPS all running OpenClaw, that's 3 nodes = $15/month. You can add or remove nodes anytime.
What happens if I cancel?
Your cloud data is retained for 30 days after cancellation, then permanently deleted. Your local self-hosted dashboard keeps working forever — you never lose that access.
