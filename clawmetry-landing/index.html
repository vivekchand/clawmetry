Sub-Agent Visibility

Your main agent spawns sub-agents. Those sub-agents run commands, burn tokens, and you have no idea what's going on. Until now. See every sub-agent, what it's doing, and what it costs.

Live Flow Graph

Channels, gateway, model, tools, infra nodes. All connected, updating live.

AI Telemetry

Token usage, response times, tool calls, cache hits, model costs. All the numbers, none of the guessing.

Session Tracking
