Frequently Asked Questions

Everything you need to know about ClawMetry.

What is ClawMetry?

ClawMetry is a free, open-source real-time observability dashboard for AI agents running on OpenClaw. It gives you instant visibility into token costs, cron jobs, sub-agent activity, memory file changes, and session history — all in a single local dashboard.

Is ClawMetry free?

Yes, completely. ClawMetry is MIT licensed and runs locally on your machine. No cloud dependency, no telemetry, no subscription. There is also an optional ClawMetry Cloud plan for teams who want a hosted multi-node dashboard.

How do I install ClawMetry?

Run pip install clawmetry then clawmetry. It auto-detects your OpenClaw workspace — no config file needed. Setup takes about 30 seconds.

What does ClawMetry monitor?

ClawMetry monitors token usage and cost per session and model, cron job status and history, sub-agent spawns and outcomes, memory file changes (SOUL.md, MEMORY.md, AGENTS.md), session transcripts, channel activity, and real-time tool call traces.

Does ClawMetry work with Claude, GPT-4, and other models?

Yes. ClawMetry works with any model your OpenClaw agents use — Claude (Anthropic), GPT-4 (OpenAI), Gemini (Google), and others. It tracks cost and token usage per model automatically.

How is ClawMetry different from LangSmith or Weights & Biases?

LangSmith and W&B are enterprise platforms built for LLM application developers. ClawMetry is purpose-built for OpenClaw users — it understands the specific concepts of channels, cron jobs, sub-agents, memory files, and the full agent runtime. It runs locally in 30 seconds with zero config.

What operating systems does ClawMetry support?

ClawMetry runs on Linux, macOS, and Windows (via WSL). It also works on Raspberry Pi and other ARM devices. Requires Python 3.8+ and an active OpenClaw installation.
